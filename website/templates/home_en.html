TX-Pi – A Raspberry Pi for fischertechnik

The aim of the TX-Pi project is to integrate the Raspberry Pi with the fischertechnik construction kits.


A complete TX-Pi setup

Read about the prerequisites required when building your own TX-Pi setup.

Software

The TX-Pi project provides software for a satisfying user experience and compatibility with the fischertechnik TXT controller.

Hardware

The TX-Pi project provides hardware for safe interfacing between the Raspberry Pi and the electrical fischertechnik components.

Cases

The TX-Pi project provides data for 3d printed cases for secure mechanical connections between the Raspberry Pi and the fischertechnik components.

New: The TX-Pi HAT is available.
