TX-Pi – A Raspberry Pi for fischertechnik

The aim of the TX-Pi project is to integrate the Raspberry Pi with the fischertechnik construction toy. The project therefore provides case designs for mechanical integration, operating system images for software integration and hardware designs for electrical integration.

The TX-Pi project provides a setup script to install the community firmware for the fischertechnik TXT on a regular Raspberry Pi.

Furthermore, we provide SD card images for an easy installation without having to deal with the install script.

New:The TX-Pi HAT ingegrates the Raspberry Pi into the 9V fischertechnik world (Vendors: Gundermann and Santjohanser).


The TX-Pi modules: Display module, (optional) I2C breakout module and Raspberry Pi module.

3D-Print data for the cases is available at the GitHub-Repository.
