Das TX-Pi-Projekt beinhaltet ein komplettes Software-Setup frü den Raspberry-Pi. Es umfasst eine einfach zu benutzende Touchscreen-Oberfläche und kommt mit vorinstallierten Anwendungen sowie der Möglichkeit, weitere Anwendungen aus den App-Stores zu laden.

{{ slideshow([ [ "p4_4inch_gui.jpg", "Pi4 running the TX Pi GUI" ], [ "launcher_320x240.png", "Touch friendly user interface" ], [ "busscanner.png", "System Tools" ], [ "3dprint.png", "Productivity Apps" ] ])}}

Die TX-Pi-Software greift viele Elemente auf, die für die Community-Firmware für den fischertechnik-TXT-Controller geschrieben wurden und sie kann die meisten Anwendungen, die dafür geschrieben wurden auch auf dem Raspberry-Pi ausführen.

Durch seine viel höhere Leistung kann der TX-Pi alles, was der TXT-Controller kann und vieles mehr. Durch die Verwendung von Python als bevorzugte Programmierumgebung sind Anfänger wie Fortgeschrittene schnell in der Lage, eigene Anwendungen zu schreiben.

Erste Schritte

Zur Zeit gibt es zwei Wege, eine TX-Pi-Softwareumgebung zu installieren:

  • Die Verwendung vorgefertigterSD-Karten-Abbilder verspricht eine einfache Installtion.
  • Die manuelle Installation bei Verwendung des Setup-Script eignet sich für erfahrene Benutzer.
